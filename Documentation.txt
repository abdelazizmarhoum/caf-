# **CafÃ© Ordering System - Complete Project Documentation**

## **Project Overview**
A modern, efficient cafÃ© ordering system that replaces traditional paper-based ordering with a digital, real-time solution. The system consists of three interconnected interfaces: **Customer Mobile Interface**, **Kitchen Staff Dashboard**, and **Manager Administration Panel**. Customers scan QR codes at their tables to order, kitchen staff receive and prepare orders in real-time, and managers have complete control over the system.

1. Customer sits at a table (e.g., Table 5)

4. **Table Confirmation Screen** appears:

3. **Shopping Cart**:

#### **Step 3: Order Confirmation**
1. Review order summary

3. **System actions**:
   - Locks table (cannot order again until current order complete)
   - Sends order to kitchen dashboard
   - Plays notification sound in kitchen
   - Redirects customer to waiting page

## **Security Measures**

1. **Table Security**:
   - QR codes encode table numbers only
   - Session validation on each request
   - Table confirmation step
   - One active order per table enforcement

2. **Authentication**:
   - JWT tokens with expiration
   - Password hashing with bcrypt
   - Role-based access control
   - Session timeout for kitchen staff

3. **Data Security**:
   - Input validation on all endpoints
   - No customer PII storage
   - HTTPS enforcement
